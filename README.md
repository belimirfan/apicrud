🚀 Employee CRUD Application

A full-stack CRUD (Create, Read, Update, Delete) web application built using:

Node.js

Express.js

MongoDB Atlas

REST API

HTML, CSS, JavaScript

This project allows users to add, edit, delete, and view employee records stored in MongoDB Atlas.
📌 Features

➕ Add new employee

📋 View all employees

✏️ Edit employee details

❌ Delete employee

🌐 REST API backend

☁️ MongoDB Atlas cloud database

🛠️ Technologies Used

Node.js

Express.js

MongoDB Atlas

Mongoose

HTML5

CSS3

JavaScript (Fetch API)

⚙️ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/belimirfan/apicrud.git
cd apicrud
2️⃣ Install Dependencies
npm install
3️⃣ Create .env File

Create a file named .env in root folder and add:

PORT=5000
MONGO_URI=your_mongodb_atlas_connection_string
4️⃣ Run the Server
npm start

Open browser:

http://localhost:5000
